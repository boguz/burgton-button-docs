#

Introduction

<burgton-button> is a small customisable lit-element burger menu that you can use in your projects.

Amongst other features, it allows you to choose from different animations, customise colours, animation speed and event toggle classes on multiple targets on the page.

#

Getting started

You can use the <burgton-button> element on your project in 3 simple steps:

1. Install the <burgton-button> on your project via NPM (or download it here)

npm install burgton-button

2. Import the <burgton-button> module on your project

<script type="module" src="node_modules/burgton-button/index.js"></script>

3. Add the <burgton-button> element to you page and customize to your taste

<burgton-button type="astronaut"></burgton-button>

API

Looking for how to customize your <burgton-button>? You came to the right place...

Attributes

TYPE [attribute, String, optional]

The type attribute defines the button animation.

Valid values: default, arrow-left, arrow-right, arrow-to-arrow, astronaut, async, crossed, ellipsis, fall-down, minus, rotator, spin, underlined, vertical-rotator, zoom Default value: default

Example:

<burgton-button type="astronaut"></burgton-button>

LABEL [attribute, String, optional]

The label attribute defines the text that will be show with the button.

If you don't add a label attribute to the element, no label will be shown.

To position the label you can use the labelPosition attribute.

Valid values: Any String Default value: no attribute

Example:

<burgton-button label="Open" labelPosition="bottom"></burgton-button>

LABEL POSITION [attribute, String, optional]

The labelPosition attribute defines the position of the label text in relation to the button.

If you don't add a label attribute to the element, no label will be shown.

Valid values: top, bottom, right, left Default value: no attribute

Example:

<burgton-button label="Menu" labelPosition="right"></burgton-button>

TARGET SELECTORS [attribute, valid CSS Selectors (comma separated), optional]

Very often we want to toggle some other element when we click on a button. You can do this yourself, of you can use the targetSelector and the targetClasses attributes and let the burgton-button take care of that for you.

The targetSelectors attribute defines on which elements you want to toggle some class.

Valid values: CSS Selector String Default value: no attribute

Example:

<burgton-button targetSelectors="#circle-one, .circle-two" targetClasses="active, colorful"></burgton-button>

TARGET CLASSES [attribute, String, optional]

The targetClasses attribute defines the class that should be toggled on the selected element.

The targetClasses is used in combination with the targetSelectors attribute.

The number of selectors (in the targetSelectors) and classes (in the targetClasses) must match.

Valid values: String Default value: no attribute

Example:

<burgton-button targetSelectors="#circle-three" targetClasses="active"></burgton-button>
