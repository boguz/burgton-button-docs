#

Introduction

<burgton-button> is a small customisable lit-element burger menu that you can use in your projects.

Amongst other features, it allows you to choose from different animations, customise colours, animation speed and event toggle classes on multiple targets on the page.

#

Getting started

You can use the <burgton-button> element on your project in 3 simple steps:

1. Install the <burgton-button> on your project via NPM (or download it here)

npm install burgton-button

2. Import the <burgton-button> module on your project

<script type="module" src="node_modules/burgton-button/index.js"></script>

3. Add the <burgton-button> element to you page and customize to your taste

<burgton-button type="astronaut"></burgton-button>

API

Looking for how to customize your <burgton-button>? You came to the right place...

Attributes

TYPE [attribute, String, optional]

The type attribute defines the button animation.

Valid values: default, arrow-left, arrow-right, arrow-to-arrow, astronaut, async, crossed, ellipsis, fall-down, minus, rotator, spin, underlined, vertical-rotator, zoom Default value: default

Example:

<burgton-button type="astronaut"></burgton-button>

LABEL [attribute, String, optional]

The label attribute defines the text that will be show with the button.

If you don't add a label attribute to the element, no label will be shown.

To position the label you can use the labelPosition attribute.

Valid values: Any String Default value: no attribute

Example:

<burgton-button label="Open" labelPosition="bottom"></burgton-button>

LABEL POSITION [attribute, String, optional]

The labelPosition attribute defines the position of the label text in relation to the button.

If you don't add a label attribute to the element, no label will be shown.

Valid values: top, bottom, right, left Default value: no attribute

Example:

<burgton-button label="Menu" labelPosition="top"></burgton-button>

TARGET SELECTORS [attribute, valid CSS Selectors (comma separated), optional]

Very often we want to toggle some other element when we click on a button. You can do this yourself, of you can use the targetSelector and the targetClasses attributes and let the burgton-button take care of that for you.

The targetSelectors attribute defines on which elements you want to toggle some class.

Valid values: CSS Selector String Default value: no attribute

Example:

<burgton-button targetSelectors="#circle-one, .circle-two" targetClasses="active, colorful"></burgton-button>

TARGET CLASSES [attribute, String, optional]

The targetClasses attribute defines the class that should be toggled on the selected element.

The targetClasses is used in combination with the targetSelectors attribute.

The number of selectors (in the targetSelectors) and classes (in the targetClasses) must match.

Valid values: String Default value: no attribute

Example:

<burgton-button targetSelectors="#circle-three" targetClasses="active"></burgton-button>

NO ANIMATION [attribute, optional]

The noAnimation attribute disables the button animation.

Example:

<burgton-button noAnimation></burgton-button>

DEBUG [attribute, optional]

The debug attribute enables the debug mode.

When active, the debug mode will print on the console some information about the <burgton-button>, which may help you find any problems.

Example:

<burgton-button debug></burgton-button>

Styles

<burgton-button> exposes several CSS custom properties that you can use to further customize it.

LINE COLOR [css custom property, String with css color value, optional]

With the --burgton-button-line-color property you can change the color of the button lines when it is inactive.

Valid values: CSS color values, like white or #FF0000 Default value: #66cc99

Example:

--burgton-button-line-color: tomato;

LINE COLOR ACTIVE [css custom property, String with css color value, optional]

With the --burgton-button-line-color-active property you can change the color of the button lines when it is active.

Valid values: CSS color values, like white or #FF0000 Default value: #66cc99

Example:

--burgton-button-line-color-active: purple;

ANIMATION DURATION [css custom property, String with css time value, optional]

With the --burgton-button-animation-duration property you set the animation duration.

Valid values: CSS time value, like .5s or 250ms Default value: 500ms

Example:

--burgton-button-animation-duration: 1s;

ANIMATION DELAY [css custom property, String with css time value, optional]

With the --burgton-button-animation-delay property you set a delay before the animation starts.

Valid values: CSS time value, like 150ms or .5s Default value: 0s

Example:

--burgton-button-animation-delay: 1s;

EASING [css custom property, String with css easing value, optional]

With the --burgton-button-easing property you set the easing function for thr button animation.

Valid values: CSS easing value, like ease-in or cubic-bezier(0, 0, 1, 1) Default value: ease-in-out

Example:

--burgton-button-easing: linear;

SCALE [css custom property, Number, optional]

With the --burgton-button-scale you can scale the button up or down.

Valid values: Number, like .75 or 1.3 Default value: 1

Example:

--burgton-button-scale: 1.5;

BORDER RADIUS [css custom property, String, optional]

With the --burgton-button-border-radius property you can make the line corners rounder.

Valid values: CSS border radius value, like 2px Default value: 0px

Example:

--burgton-button-border-radius: 2px;

LABEL FONT SIZE [css custom property, String, optional]

With the --burgton-button-label-font-size property you can set the font size for the button label.

Valid values: CSS font size value, like 14px or 1.25rem Default value: 14px

Example:

--burgton-button-label-font-size: 10px;

LABEL FONT COLOR [css custom property, String, optional]

With the --burgton-button-label-font-color property you can set the color for the button label.

Valid values: CSS color value, like white or #FF0000 Default value: #999999

Example:

--burgton-button-label-font-color: tomato;

LABEL FONT COLOR ACTIVE[css custom property, String, optional]

With the --burgton-button-label-font-color-active property you can set the color for the button label when active.

Valid values: CSS color value, like white or #FF0000 Default value: #999999

Example:

--burgton-button-label-font-color-active: purple;

LABEL FONT WEIGHT[css custom property, String, optional]

With the --burgton-button-label-font-weight property you can set the font-weight for the button label.

Valid values: CSS font-weight value, like 100 Default value: inherit

Example:

--burgton-button-label-font-weight: 600;

LABEL TEXT TRANSFORM [css custom property, String, optional]

With the --burgton-button-label-text-transform property you can set the text-transform for the button label.

Valid values: CSS text-transform value, like lowercase Default value: uppercase

Example:

--burgton-button-label-text-transform: capitalize;
